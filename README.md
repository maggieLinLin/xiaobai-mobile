# 小白機 - 虛擬手機模擬器

一個基於 Web 的手機模擬器，具有完整的 UI 和多種功能。

## 功能特色

- 📱 手機界面模擬（414×736）
- 🎵 音樂播放器
- 📅 日曆與備忘錄
- 💬 AI 聊天助手（需配置 API）
- 🎨 多主題切換（玉桂狗、Hello Kitty、庫洛米、美樂蒂）
- 🔤 自定義字體
- 🎨 自定義壁紙和 CSS

## 在線預覽

訪問：https://你的用戶名.github.io/xiaobai-mobile/

## 本地運行

1. 克隆倉庫
```bash
git clone https://github.com/你的用戶名/xiaobai-mobile.git
```

2. 打開 `index.html` 即可使用

## 使用說明

- 下拉或點擊灵動島打開設置
- 左右滑動切換頁面
- 點擊應用圖標打開應用

## 技術棧

- HTML5
- CSS3
- Vanilla JavaScript
