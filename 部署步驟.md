# Cloudflare Workers 部署步驟（5分鐘完成）

## 第一步：註冊 Cloudflare 帳號
1. 訪問：https://dash.cloudflare.com/sign-up
2. 輸入郵箱和密碼註冊（完全免費，無需信用卡）
3. 驗證郵箱

## 第二步：創建 Worker
1. 登錄後訪問：https://dash.cloudflare.com/
2. 點擊左側菜單「Workers & Pages」
3. 點擊「Create application」按鈕
4. 選擇「Create Worker」
5. 輸入名稱（例如：music-api）
6. 點擊「Deploy」

## 第三步：編輯代碼
1. 部署後會自動跳轉，點擊「Edit code」按鈕
2. 刪除編輯器中的所有代碼
3. 打開 `worker-simple.js` 文件
4. 複製所有內容
5. 粘貼到 Cloudflare 編輯器中
6. 點擊右上角「Save and Deploy」

## 第四步：獲取 URL
部署成功後，你會看到類似這樣的 URL：
```
https://music-api.你的用戶名.workers.dev
```
複製這個 URL！

## 第五步：更新代碼
打開 `app.js`，找到 `searchMusic` 函數，將第一個 API 改為：

```javascript
const apis = [

```

3. 搜索音樂測試

## 如果遇到問題
1. 確保 Worker 代碼正確複製
2. 確保 URL 正確（不要有多餘的空格）
3. 在瀏覽器按 F12 查看 Console 錯誤信息
