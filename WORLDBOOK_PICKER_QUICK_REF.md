# 🎯 世界书可视化选择器 - 快速参考

## 🚀 30秒快速上手

### 在聊天设置中选择

```
1. LINEE → 聊天室 → ⋯ → 聊天设置
2. 点击 "全局世界书" 或 "局部世界书"
3. 在世界书 App 中点击卡片选择
4. 点击右上角 "✓ 确定"
5. 完成!
```

### 在好友资料中选择

```
1. LINEE → 好友 → 点击 AI 角色
2. 滚动到 "关联世界书" 卡片
3. 点击选择按钮
4. 在世界书 App 中点击卡片选择
5. 点击右上角 "✓ 确定"
6. 完成!
```

## ⌨️ 操作说明

| 操作 | 效果 |
|------|------|
| 点击未选中的卡片 | ✅ 选中 (显示 ☑️) |
| 点击已选中的卡片 | ❌ 取消选中 (显示 ☐) |
| 点击右上角 "✓" | 💾 保存并返回 |
| 点击左上角 "×" | 🚫 取消并返回 |

## 💡 实用技巧

### 多选技巧

**连续点击:**
```
点击 A → 已选 1
点击 B → 已选 2
点击 C → 已选 3
```

**取消选择:**
```
再次点击 B → 已选 2 (取消了 B)
```

### 清空所有选择

```
方法 1: 点击所有已选中的卡片
方法 2: 点击 "×" 取消,重新进入选择器
```

## ❓ 常见问题

### Q: 点击卡片没反应?

**A:** 检查是否在选择器模式 (顶部有 "已选 X 个")

### Q: 如何取消所有选择?

**A:** 点击所有已选中的卡片,或点击 "×" 重新开始

### Q: 选择后如何保存?

**A:** 点击右上角蓝色 "✓ 确定" 按钮

### Q: 如何返回上一页?

**A:** 点击左上角 "×" 或完成选择后自动返回
