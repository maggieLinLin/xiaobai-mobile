# Linee App 整合說明

## 📱 整合完成

Linee 應用已成功整合進小白機虛擬手機模擬器中，作為一個獨立的 App 存在。

## ✨ 功能特色

### UI 設計
- **極簡 LINEE 風格**：主色 #A0D8EF，輔助色 #E8F6FA、#B8DFF2
- **線性極細圖標**：1.5px 線條，iOS outline 風格
- **圓角設計**：12-20px 圓角，大量留白
- **完美適配**：414×736 手機模擬器尺寸

### 核心功能
1. **個人資料區**
   - 頭像顯示與編輯
   - 可編輯狀態訊息
   - 三個情境卡片（日常、工作、休閒）

2. **搜尋功能**
   - 搜尋好友和群組

3. **好友管理**
   - 好友列表（可展開/收起）
   - 添加新好友
   - 顯示好友狀態

4. **群組管理**
   - 群組列表（可展開/收起）
   - 建立新群組
   - 顯示群組成員數

5. **特殊功能區**
   - 當月壽星
   - 我的最愛

6. **底部導航**
   - 主頁
   - 聊天
   - 足跡
   - 日記
   - 錢包

## 🎯 使用方式

1. **開啟 Linee App**
   - 在主頁面點擊 "Linee" 應用圖標
   - App 會在手機模擬器內全屏顯示

2. **返回主頁**
   - 點擊左上角的返回按鈕（←）

3. **添加好友**
   - 點擊右上角的 ➕ 按鈕
   - 選擇「加入好友」
   - 輸入好友名字並確認

4. **建立群組**
   - 點擊右上角的 ➕ 按鈕
   - 選擇「建立群組」
   - 輸入群組名稱並確認

5. **展開/收起列表**
   - 點擊「好友」或「群組」標題
   - 列表會自動展開或收起

## 🔧 技術細節

### CSS 架構
- 使用 CSS 變數定義主題色
- 所有樣式都加上 `#linee-app-content` 前綴，避免與主程式衝突
- 響應式設計，完美適配手機尺寸

### JavaScript 功能
- 獨立的初始化函數 `initLineeApp()`
- 防止重複初始化機制
- 全局函數註冊，支援 HTML onclick 事件
- 動態渲染好友和群組列表

### 數據管理
- 使用 JavaScript 陣列儲存好友和群組數據
- 支援動態添加和更新
- 未來可擴展為 localStorage 持久化

## 📝 注意事項

1. **樣式隔離**：所有 Linee 樣式都使用 `#linee-app-content` 前綴，確保不會影響其他頁面

2. **函數命名**：所有 Linee 相關函數都以 `linee` 或 `Linee` 開頭，避免命名衝突

3. **初始化時機**：Linee App 只在首次打開時初始化，避免重複執行

4. **事件處理**：使用事件委託和 stopPropagation 避免事件冒泡衝突
