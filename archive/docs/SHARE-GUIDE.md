# 分享小白機給朋友 - 使用指南

## 當前狀態

✅ **已經可以分享！** 您的小白機已內建多個音樂 API，別人只需要 HTML 文件就能使用。

## 分享方式

### 方式 1：直接分享文件（推薦）

**優點**：
- ✅ 無需部署
- ✅ 立即可用
- ✅ 完全離線（除了音樂搜索）

**步驟**：
1. 壓縮整個 `xiaobai-mobile` 文件夾
2. 分享給朋友
3. 朋友解壓後打開 `index.html`
4. 音樂搜索會自動使用內建的 3 個 API

**內建 API**：
- API 1: Railway 音樂 API
- API 2: Meting API (網易雲)
- API 3: Cyril Studio API

### 方式 2：部署到 Vercel（可選）

**優點**：
- ✅ 分享一個網址即可
- ✅ 自動更新
- ✅ 全球 CDN 加速

**步驟**：
1. 按照 `DEPLOY-GUIDE.md` 部署到 Vercel
2. 獲得網址如：`https://xiaobai-mobile.vercel.app`
3. 分享網址給朋友

## 音樂搜索工作原理

```
用戶打開 HTML → 搜索音樂 → 自動嘗試 3 個 API → 返回最快結果
```

**無需**：
- ❌ 用戶自己部署
- ❌ 用戶配置 API
- ❌ 用戶知道技術細節

**只需**：
- ✅ 打開 HTML 文件
- ✅ 有網絡連接

## 常見問題

### Q: 朋友需要安裝什麼嗎？
A: 不需要！只需要瀏覽器。

### Q: 音樂搜索會失敗嗎？
A: 有 3 個備用 API，一個失敗會自動嘗試下一個。

### Q: 可以離線使用嗎？
A: 大部分功能可以（日曆、備忘錄、本地音樂），但音樂搜索需要網絡。

### Q: 如何讓音樂搜索更穩定？
A: 部署到 Vercel，使用您自己的 API 端點。
