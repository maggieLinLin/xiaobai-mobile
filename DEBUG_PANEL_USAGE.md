# 🐛 调试面板使用指南

## 快速开始

### 打开调试面板

1. 在任意页面,查看右上角
2. 点击红色按钮 **🐛 Debug**
3. 调试面板会从右侧滑出

![调试面板位置示意图]
```
┌─────────────────────────┐
│ [🐛 Debug]  ← 点这里      │

│   [调试窗口]              │
│   - 当前聊天设置           │
│   - 当前角色信息           │
│   - 最后的 System Prompt  │
└─────────────────────────┘
```

## 常见问题排查

### ❓ 问题1: AI 没有按照我的人设回复

**排查步骤:**
1. 打开调试面板
2. 查看 "当前角色信息" 部分
3. 确认 `外貌` 和 `背景` 是否有内容

**如果显示空白或默认值:**
→ 说明角色创建时没有填写完整,请重新编辑角色

---

### ❓ 问题2: AI 还是一长串不断句

**排查步骤:**
1. 打开调试面板
2. 查看 "当前聊天设置"
3. 确认 `线下模式` 是否 **关闭**

**如果显示 ✅ 开启:**
→ 说明你在线下模式,需要在聊天设置中关闭

**然后查看 "最后发送的 System Prompt":**
→ 确认是否包含 "★★★ 线上模式·绝对法则"

---

### ❓ 问题3: AI 还是每句都加 Emoji

**排查步骤:**
1. 打开调试面板
2. 查看 "最后发送的 System Prompt"
3. 搜索关键词 "Emoji 使用禁令"

**如果找不到这段内容:**
→ 说明 Prompt 没有正确更新,请刷新页面

**如果能找到但 AI 还是乱加:**
→ 可能是模型太弱,建议:
  - 提高 Temperature 到 0.9
  - 或换用更强的模型 (GPT-4)

---

### ❓ 问题4: 调试面板显示"尚未生成 Prompt"

**原因:** 你还没有发送过消息

**解决方法:**
1. 在聊天中发送一条消息
2. 等待 AI 回复
3. 再次打开调试面板,就能看到 Prompt 了

## 实时更新

调试面板会每秒自动刷新 (如果保持打开状态)。

你可以:
1. 打开调试面板
2. 修改聊天设置 (例如开启/关闭线下模式)
3. 观察调试面板,确认设置已更新

---

## 关闭面板

点击调试窗口右上角的 **×** 按钮即可关闭。

---

## 高级技巧

### 验证世界书是否生效

1. 在世界书中添加条目,例如:
   - 关键词: "小白机"
   - 内容: "一部神奇的智能手机模拟器"

2. 在聊天中提到 "小白机"

3. 打开调试面板,查看 "最后发送的 System Prompt"

4. 搜索 "【世界观信息】"

5. 确认是否包含你添加的内容

---

### 对比不同模式的 Prompt

1. 在**线上模式**下发送一条消息
2. 打开调试面板,复制 Prompt (Ctrl+C)
3. 关闭线上模式,切换到**线下模式**
4. 再发送一条消息
5. 对比两次的 Prompt 差异

**你会发现:**
- 线上模式: 包含 "手机通讯软件" + "断句规则" + "Emoji 禁令"
- 线下模式: 包含 "面对面沉浸式" + "网文节奏" + "慢镜头技巧"

---

## 注意事项

⚠️ 调试面板是**只读**的,你不能直接修改 Prompt。

如果发现 Prompt 有问题,需要:
1. 修改角色信息 (在角色编辑页面)
2. 修改聊天设置 (在聊天设置页面)
3. 修改世界书 (在世界书管理页面)

然后重新发送消息,调试面板会自动显示新的 Prompt。
